xvision.transforms.umeyama.umeyama(src, dst, estimate_scale=True)[source]¶

Estimate N-D similarity transformation with or without scaling. :param src: Source coordinates. :type src: (M, N) array :param dst: Destination coordinates. :type dst: (M, N) array :param estimate_scale: Whether to estimate scaling factor. :type estimate_scale: bool

Returns

T – The homogeneous similarity transformation matrix. The matrix contains NaN values only if the problem is not well-conditioned.

Return type

(N + 1, N + 1)

References

1

“Least-squares estimation of transformation parameters between two point patterns”, Shinji Umeyama, PAMI 1991, DOI: 10.1109/34.88573
